# MyShellScripts

一些自用的 Shell 脚本

## 脚本列表

### downloaded-videos-auto-sync
Linux 视频文件自动同步工具，监控下载目录变化并自动同步到指定位置。

- 实时文件监控
- 智能重试机制
- systemd 服务支持

### powershell-auto-proxy
PowerShell 代理自动配置工具，智能检测代理状态并自动切换。

- 自动检测代理可用性
- 支持 HTTP/HTTPS 和 SOCKS5
- 启动时智能提示

## 使用方法

每个脚本目录都有详细的 README 文档，请查看对应目录。
